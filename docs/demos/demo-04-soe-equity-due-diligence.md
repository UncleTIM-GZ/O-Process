## 三、国企特殊风险扫描

**Tool:** `search_process` + `get_kpi_suggestions`

```json
{ "query": "regulatory compliance state-owned enterprise risk", "lang": "en", "limit": 10 }
```

在标准尽调清单基础上，叠加国企特有风险项，逐项判断风险等级：

| 序号 | 核查项目 | 风险等级 | 判断依据 | PCF 对标 |
|------|---------|---------|---------|----------|
| **①** | **国有产权转让是否需经国资委审批或备案** | **HIGH** | 省属二级子公司转让 51% 控股权属于重大产权变动，须报省级国资委审批。若标的所在市另有授权监管，须同时确认市级国资委是否具有审批权限。审批层级判断错误将导致交易无效。 | `11.2.2.2` 识别适用的监管要求 |
| **②** | **是否存在未解除的政策性负担** | **HIGH** | 省属国企二级子公司（制造业）历史上大概率承担过政策性安置义务、企业办社会职能。若存在未剥离的离退休人员统筹外费用、厂办学校/医院等负担，直接影响标的估值及交割条件设定。 | `7.4.1` 管理劳资关系 + `7.6.3` 管理退休 |
| **③** | **党委前置程序是否已履行** | **HIGH** | 依据《中国共产党国有企业基层组织工作条例》第15条，国有企业重大经营管理事项须经党委（党组）研究讨论后再由董事会或经理层作出决定。控股权转让属于重大事项，未经党委前置审议的股权转让决议存在效力瑕疵。 | `9.8.1` 建立内部控制、政策和程序 |
| **④** | **土地性质核查（划拨地是否可随股权转让过户）** | **MEDIUM** | 股权转让不直接触发划拨土地使用权变更登记，但取得控股权后变更实际控制人时，部分地方自然资源部门要求补缴出让金。须查明标的名下所有地块的用地性质及当地政策口径。 | `10.1.1` 制定资产战略和长期愿景 |
| **⑤** | **在职及退休职工安置方案是否已形成书面文件** | **HIGH** | 32号令要求国有产权转让涉及职工合法权益的，须制定职工安置方案并经职工代表大会审议通过。未完成此程序即进场尽调，存在后续因职工安置方案未过会导致交易中止的风险。 | `7.6.2` 管理离职 + `7.4.2` 管理集体谈判流程 |

## 四、尽调工作底稿框架

### 模块 1：国资合规

| 项目 | 内容 |
|------|------|
| **核查目标** | 确认国有产权转让已取得全部法定审批，产权交易程序合法有效 |
| **所需文件清单** | ① 省国资委关于本次股权转让的批复/备案文件 ② 资产评估报告及国资委核准/备案文件 ③ 产权交易所挂牌公告（或协议转让豁免审批文件） ④ 职工代表大会关于职工安置方案的决议 ⑤ 标的公司党委关于本次转让的前置审议决议 |
| **访谈对象** | 集团国资运营部负责人、标的公司董事长/总经理 |
| **核查方式** | 文件审查 + 访谈 |
| **主要风险点** | ① 审批层级判断错误导致交易无效 ② 评估基准日与签约日间隔超过一年导致评估报告失效 |
| **风险等级** | **HIGH** |

### 模块 2：主体资格

| 项目 | 内容 |
|------|------|
| **核查目标** | 确认标的公司法人主体合法存续，股权权属清晰，不存在转让障碍 |
| **所需文件清单** | ① 营业执照（三证合一）及公司章程（现行有效版本） ② 历次股权变更的工商变更登记档案 ③ 股东会/董事会/监事会决议（近三年） ④ 行业资质证书、安全生产许可证 ⑤ 国家企业信用信息公示系统查询结果 |
| **访谈对象** | 标的公司法务经理/公司秘书 |
| **核查方式** | 文件审查 |
| **主要风险点** | ① 章程中存在限制控股权变更的特殊条款 ② 行业许可证因控制权变更须重新申请 |
| **风险等级** | **MEDIUM** |

### 模块 3：土地资产

| 项目 | 内容 |
|------|------|
| **核查目标** | 确认标的名下不动产权属清晰，不存在权利瑕疵，划拨地处置方案明确 |
| **所需文件清单** | ① 不动产权证书（全部地块及房屋） ② 在建工程规划许可证、施工许可证 ③ 土地出让合同或划拨决定书 ④ 不动产登记查询结果（查封/抵押/冻结信息） ⑤ 当地自然资源部门关于划拨土地补缴出让金的政策文件 |
| **访谈对象** | 标的公司基建部/资产管理部负责人 |
| **核查方式** | 文件审查 + 现场查看 |
| **主要风险点** | ① 划拨土地补缴出让金金额重大，影响交易对价 ② 在建工程缺少规划/施工许可，存在被责令拆除风险 |
| **风险等级** | **HIGH** |

## 五、进场前必须确认的 3 个前提条件

**Tool:** `search_process`

```json
{ "query": "regulatory approval prerequisite due diligence", "lang": "en", "limit": 10 }
```

**若以下条件任一未满足，建议暂缓进场或书面保留意见：**

| 前提条件 | 核查依据 | PCF 对标 | 未满足时的处置建议 |
|---------|---------|----------|------------------|
| **前提 1：省国资委已出具关于本次股权转让的原则性批复或授权审批文件** | 32号令第5条要求国有产权转让须经履行出资人职责的机构同意。无批复即启动尽调，全部工作成果存在被推翻的风险。 | `11.2.2.2` 识别适用的监管要求 | 暂缓进场，书面函告委托方：在取得国资委批复前不具备启动尽调的前置条件。 |
| **前提 2：标的公司已完成资产审计和评估，评估报告已报国资委核准/备案** | 32号令第19条、第21条要求评估须先行完成且经核准。未经评估的标的无法确定挂牌底价，尽调结论将缺少估值锚点。 | `9.3.4` 执行财务报告 + `11.2.2.8` 编制合规报告 | 暂缓进场，或在委托函中明确注明"本次尽调在评估报告出具前进行，最终结论以评估结果为准"。 |
